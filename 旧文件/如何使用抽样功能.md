# 金铲铲数据抽样功能使用指南

## 概述

`01sampling.py` 脚本提供了两种方式来进行数据抽样：
1. **命令行模式**：通过终端命令执行
2. **程序化模式**：在Python代码中直接调用

## 功能特点

✅ 自动避免重复抽样（检查现有样本文件）  
✅ 智能文件命名（自动递增序号）  
✅ 字符长度过滤（可选）  
✅ 完整的错误处理和日志记录  
✅ 严格遵循data-management规范  

## 使用方法

### 方法1: 命令行模式

```bash
# 基本抽样：抽取100条样本
python 01sampling.py -n 100

# 带字符长度过滤：抽取50条样本，跳过评论少于200字符的数据
python 01sampling.py -n 50 -l 200
```

### 方法2: 程序化调用模式

在你的Python代码中直接调用抽样功能：

```python
# 导入抽样函数（需要确保01sampling.py在同一目录）
import importlib.util
import sys

# 动态导入01sampling模块
spec = importlib.util.spec_from_file_location("sampling", "01sampling.py")
sampling_module = importlib.util.module_from_spec(spec)
spec.loader.exec_module(sampling_module)

# 获取抽样函数
sample_data_programmatic = sampling_module.sample_data_programmatic

# 基本抽样
result = sample_data_programmatic(sample_size=100)

if result['success']:
    print(f"✅ 抽样成功！")
    print(f"   样本文件: {result['sample_file']}")
    print(f"   抽样数量: {result['sample_count']}")
    print(f"   消息: {result['message']}")
    
    # 可以访问抽样数据
    sample_df = result['sample_data']
    print(sample_df.head())
else:
    print(f"❌ 抽样失败: {result['message']}")

# 带字符长度过滤的抽样
result = sample_data_programmatic(sample_size=50, min_length=200)
```

### 方法3: 直接运行演示

如果直接运行`01sampling.py`而不提供命令行参数，会自动运行演示：

```bash
python 01sampling.py
```

这会执行多个抽样示例，展示不同的用法。

## 返回结果格式

程序化调用返回一个包含以下键的字典：

```python
{
    'success': bool,              # 是否成功
    'sample_file': str,           # 新样本文件名
    'sample_count': int,          # 实际抽样数量
    'sample_data': pd.DataFrame,  # 抽样数据
    'message': str                # 结果消息
}
```

## 文件命名规则

- 原始数据文件：`{APP_NAME}.csv`（例如：`金铲铲.csv`）
- 样本文件：`{APP_NAME}_sample0.csv`、`{APP_NAME}_sample1.csv` 等

## 数据结构要求

- 第一列：用户ID
- 第二列：用户评论
- 编码：UTF-8

## 注意事项

1. **路径管理**：所有路径通过`parameters.py`的`get_path()`接口获取
2. **去重机制**：自动检查现有样本文件，避免重复抽样同一用户ID
3. **字符过滤**：可选的评论长度过滤，跳过字符数不足的评论
4. **数据完整性**：抽样过程中保持数据的完整性和一致性

## 常见使用场景

### 场景1: 逐步抽样分析

```python
# 第一次抽样：100条用于初步分析
result1 = sample_data_programmatic(100)

# 第二次抽样：50条补充数据
result2 = sample_data_programmatic(50)

# 第三次抽样：只要长评论，用于深度分析
result3 = sample_data_programmatic(30, min_length=500)
```

### 场景2: 批量处理

```python
sample_sizes = [50, 100, 200]
results = []

for size in sample_sizes:
    result = sample_data_programmatic(size)
    results.append(result)
    if result['success']:
        print(f"批次 {size}: 成功抽样 {result['sample_count']} 条")
```

### 场景3: 条件抽样

```python
# 根据不同条件进行抽样
conditions = [
    {'size': 100, 'min_length': None, 'desc': '基础样本'},
    {'size': 50, 'min_length': 200, 'desc': '中等长度样本'},
    {'size': 30, 'min_length': 500, 'desc': '详细评论样本'}
]

for condition in conditions:
    result = sample_data_programmatic(
        condition['size'], 
        condition['min_length']
    )
    print(f"{condition['desc']}: {'成功' if result['success'] else '失败'}")
```
